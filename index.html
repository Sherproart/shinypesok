Задание 4

при стандартной отправке указанной ниже формы:

Значение a

Значение b

Значение c

  1. перехватить событие отменив отправку
  2. собрать данные полей у которых присутствует клас add и просумировать
  3. полученную сумму добавить в набор данных отдельным параметром с именем summ
  4. отправить post запрос(на backend)
    считаем что в ответ на отправку мы получили json строку
    { "status": "success", "message": ["данные сохранены", "сумма = x"] }
  5. уведомить пользователя о успешной отправке, выдав ему по отдельности полученные сообщения в виде всплывающих блоков на 3 секунды.

 

 

Решение

Для посылки и приема POST данных использована jquery 3.1.1.
Прием-передача были проверены на локальном сервере. Все работает в формате JSON в соответствии с предполагаемым вами форматом. Поля status и message выдаются в отдельных тегах. Код серверного скрипта закомментирован в html исходнике.

Для скрытия всплывающего блока использована функция setTimeout.
